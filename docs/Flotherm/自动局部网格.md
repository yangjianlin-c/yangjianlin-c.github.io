从Flotherm 2020.2 开始，提供了一个新的Feature，自动局部网格。目前这个功能还是Beta 阶段。

首先，在工具栏勾选 'Beta Features' ，以打开这个功能。

![[assets/Pasted image 20221024093537.png]]

在System Grid 设置中，勾选 Automatic Gredding
