

# VL6180X测距
## VL6180X模块


VL6180X模块引脚

- VIN：电源电压（3-5V），芯片使用2.8V的电压，模块上有一个LDO将输入电压转换为2.8V。
- 2V8：2.8V稳压器输出，最大输出电流100mA。
- GND：地线
- GPIO1：可编程中断输出
- SHDN：传感器的关闭引脚。默认情况下，它被拉高。有一个电平转换二极管，因此您可以在此引脚上使用3-5V逻辑。当引脚拉低时，传感器进入关断模式。
- SCL：I2C时钟
- SDA：I2C数据
