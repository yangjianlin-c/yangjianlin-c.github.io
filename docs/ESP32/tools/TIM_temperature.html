TIM Temperature Rise

Input Parameters

Item Value Unit
Width $W$ mm
Length $L$ mm
Gap/Thickness $t$ mm
Conditicity $k$ W/K-m
Power $Q$ W

Result

Item Value Unit
Temperature Rise $\Delta T$

Hints

Tempearture rise due to thermal pad or thermal gel is: $$\Delta T = \frac{Qt}{Ak}$$ However, should be notice this formular just used for calculate the temeprature rise due to thermal pad. For testing and calculate thermal pad performance, it should also conside the contact thermal resistance between thermal pad and heatsink or devices.
